# Ustar_HP20M — Modbus RTU 透传桥接（HPTM100 压力温度传感器）

基于 STM32U575VGT6 的 Modbus RTU 透传工具，用于 PC 与 HPTM100 压力/温度传感器之间的 RS485 通信。

## 硬件平台

- **MCU:** STM32U575VGT6 (Cortex-M33, LQFP100)
- **传感器:** HPTM100 压力/温度传感器
- **通信接口:** RS485（自动方向控制模块）
- **开发环境:** STM32CubeIDE / Keil MDK-ARM

## 外设使用

| 外设 | 波特率 | 引脚 | 用途 |
|------|--------|------|------|
| USART1 | 9600 | PA9/PA10 | PC 端（USB-TTL） |
| USART3 | 9600 | PA7/PA5 | 传感器端（RS485） |

## 功能说明

### 通信流程

1. PC 通过 USART1 发送 Modbus RTU HEX 命令
2. MCU 将命令转发到 USART3（RS485 总线）
3. MCU 接收传感器响应（含 RS485 回声）
4. 通过 CRC16 校验提取有效 Modbus 响应帧
5. 仅将有效响应返回给 PC

### 示例命令

读取压力/温度寄存器的标准 Modbus RTU 命令通过串口助手以 HEX 格式发送。

## 技术特点

- **Modbus CRC16** 完整实现
- **RS485 回声处理：** `ExtractResponse` 函数扫描接收数据，通过 CRC16 验证提取有效 `[地址][0x03][字节数][数据][CRC]` 响应帧，自动跳过回声字节
- **超时控制：** 首字节等待 500ms，字节间超时 50ms
- 轮询接收 + 可配置超时机制
- 所有 17 个项目中协议解析最完整的一个，面向工业传感器实际应用
